Features

Everything you need

No configuration, no cloud dependencies, no nonsense. Just install and play.

🌐

200+ Languages

Powered by Meta's NLLB-200 distilled 600M model. Translate English subtitles to Turkish, German, French, Japanese, Korean, Arabic, and 200 more — all running locally on your GPU.

⚡

GPU Accelerated

CTranslate2 INT8 quantization + ONNX Runtime CUDA. ~40ms translation latency. Zero perceptible delay between the original subtitle and your translation.

🎮

Game-Aware Detection

Custom YOLO model trained on thousands of game screenshots detects subtitle regions with high precision. Per-game OCR profiles handle white, orange, and colored text.

🔒

100% Local & Private

Everything runs on your machine. No cloud APIs, no telemetry, no account required. Your gameplay stays on your PC.

📚

Learns As You Play

Growing memory database caches every translation locally. Repeated lines return instantly. Cloud-sync approved human corrections between sessions.

🔄

Auto-Update

EXE updates delivered via GitHub Releases. YOLO model improvements pushed over-the-air through Firebase. Always on the latest without reinstalling.
