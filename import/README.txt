# Subtitle Import Folder
# ======================
#
# PLACE EXTRACTED SUBTITLE FILES HERE.
#
# Supported Formats:
# ------------------
# 1. Plain Text (.txt, .lng.txt):
#    - One line per subtitle.
#    - Example:
#      Hello Artyom.
#      Watch out!
#
# 2. CSV (.csv):
#    - Column 1: Original Text
#    - Column 2: Translated Text (Optional, if you have a translation)
#
# How to use:
# -----------
# 1. Extract the text from your game (e.g., using 'UnpackerRedux' or 'MetroEx').
# 2. Place the .txt file in this folder.
# 3. Restart the Dublaj program.
# 4. The program will automatically read these files and use them for instant local translation.
