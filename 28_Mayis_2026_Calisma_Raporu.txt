2. OCR OPTIMIZASYONU
─────────────────────

  RapidOCR ONNX Runtime:
    - CPU: 551ms → GPU CUDA (cuDNN 9.x): 134ms

  Preprocessing pipeline:

  NVIDIA DLL kaydı:
    subtitle_detector.py modül yüklemesinde otomatik PATH kaydı

3. PIPELINE MİMARİSİ
─────────────────────

  Final pipeline:
    Ekran Görüntüsü (MSS, ~1ms)
      → YOLOv8-nano tespit (7ms)
      → Bbox birleştirme (çakışan kutular teklenir)
      → Crop + CLAHE+Otsu+Upscale preprocessing
      → Full OCR GPU (118ms)
      → Watermark/Kredi filtresi
      → TemporalFilter (2 frame onay, tek emit)
      → Translation Memory hash lookup (0.02 µs)
      → Overlay gösterimi (WDA_EXCLUDEFROMCAPTURE)

  Toplam gecikme: ~130ms (hedef <200ms) ✓
  YOLO: 7ms | OCR: 118ms | Memory: 0ms | Overlay: anlık


4. TEMPORAL FILTER İYİLEŞTİRMELERİ
───────────────────────────────────

  - "emit" eşiği: 3 frame → 2 frame (600ms → 400ms, 8 FPS'te 250ms)
  - "emitted" flag: aynı metin bir kez emit edilir, sonraki frame'ler "active"
  - Punctuation-insensitive key: "dying." ve "dying" aynı metin sayılır
  - Noise eşiği: <3 frame → <2 frame
  - Blacklist: 8 saniye sabit kalan metinler (tabela/HUD)
